# RaiseReady Child Platform Template

This is a template repository for creating white-label RaiseReady platforms.

## DO NOT MODIFY THIS REPO DIRECTLY

Changes should be made to the parent RaiseReady project, then synced here.

## What Gets Customized

When a new platform is created:
1. config/client.ts is regenerated with client branding
2. Environment variables are set in Vercel
3. Supabase database is created with full schema
